This is the MTSXtremIO PowerShell Module.

This Module provides PowerShell access to the EMC XtremIO REST API.

The only windows client dependency is on PowerShell 4.0

Notes:
--------------------------------------------------------------------------
This module relies on the EMC XtremIO REST API. It was developed using
version 3.0 of the API. Changes to the API could affect operation of this 
module. 

Please test any use of this module thoroughly before deploying to 
production environments.

Usage examples of the script based cmdlets are found in the module help
Example: 
	import-module MTSXtremIO
	get-help get-XIOCluster
